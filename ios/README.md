# 📍 أماكني — نسخة آيفون (iOS)

نسخة Swift/SwiftUI كاملة توازي تطبيق الأندرويد بنفس المزايا بالضبط: حفظ الموقع الحالي،
تسميته، مشاركته، حفظ عدة أماكن، والتنقل إليها — **مجاناً بالكامل**.

## 🛠️ التقنيات المستخدمة (كلها مجانية ومن آبل مباشرة)

| الغرض | التقنية |
|---|---|
| اللغة والواجهة | Swift + SwiftUI (iOS 17 فأحدث) |
| الموقع | Core Location (مجاني، جزء من iOS) |
| قاعدة البيانات | SwiftData (تخزين محلي دائم على الجهاز) |
| الملاحة والمشاركة | روابط خرائط آبل (`maps.apple.com`) — بدون أي مفتاح API |
| توليد مشروع Xcode | [XcodeGen](https://github.com/yonaskolb/XcodeGen) — يحوّل `project.yml` إلى `.xcodeproj` تلقائياً |
| البناء | GitHub Actions على جهاز macOS سحابي مجاني |

## 📁 الهيكل

```
ios/
├── project.yml                  إعدادات XcodeGen (بديل عن .xcodeproj اليدوي)
└── Sources/
    ├── SavedPlacesApp.swift     نقطة الانطلاق
    ├── Models/Place.swift       نموذج SwiftData
    ├── Services/
    │   ├── LocationManager.swift   تحديد الموقع (async/await)
    │   └── MapsLauncher.swift      روابط التنقل والمشاركة
    └── Views/                   شاشات SwiftUI
```

لا يوجد ملف `.xcodeproj` مرفوع على GitHub عمداً — يُولَّد تلقائياً في كل مرة عبر
`xcodegen generate` أثناء البناء، بدل حفظ ملف Xcode المعقّد يدوياً.

## ▶️ كيف يعمل البناء التلقائي

عند أي رفع (`git push`) لهذا المجلد، `.github/workflows/build-ios.yml` يقوم بـ:
1. تثبيت XcodeGen على جهاز macOS سحابي
2. توليد `SavedPlaces.xcodeproj` من `project.yml`
3. بناء التطبيق **بدون توقيع** (`CODE_SIGNING_ALLOWED=NO`) — التوقيع يصير على جهازك لاحقاً
4. تحزيمه كملف `SavedPlaces.ipa` وإتاحته للتحميل من تبويب **Actions**

## 📲 تثبيته على آيفونك (مجاناً، بدون حساب مطوّر مدفوع)

1. ثبّت **SideStore** على آيفونك (يحتاج كمبيوتر ويندوز أو ماك **مرة واحدة فقط** للإعداد الأولي):
   https://sidestore.io
2. حمّل ملف `SavedPlaces-ios-ipa` من تبويب Actions في GitHub (بعد اكتمال البناء بعلامة ✅)
3. افتح الملف عبر SideStore — يوقّعه ويثبّته بحساب آبل المجاني تبعك مباشرة
4. يتجدد تلقائياً بالخلفية كل 7 أيام بدون أي تدخل منك

## ⚠️ تبسيطات مقصودة (يمكن تطويرها لاحقاً)

- **لا توجد أيقونة تطبيق مخصصة بعد** — سيظهر بأيقونة افتراضية فارغة؛ لا تؤثر على عمل التطبيق إطلاقاً.
- **بدون خريطة مصغّرة داخل التطبيق** — الملاحة تفتح تطبيق خرائط آبل مباشرة بدل عرض خريطة مضمّنة، لتبسيط المشروع الأول وتقليل احتمالية الأخطاء.
- الحد الأقصى 3 تطبيقات "جانبية" مثبتة بنفس الوقت بحساب آبل المجاني — قيد من آبل نفسها، غير قابل للتجاوز إلا بحساب مطوّر مدفوع (99$/سنة).
