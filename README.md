# 📍 أماكني — تطبيق حفظ الأماكن (Saved Places)

تطبيق أندرويد بسيط وسريع يتيح لك حفظ أي موقع تقف فيه بضغطة زر، وإعطاءه اسماً تتذكره به،
ثم مشاركته أو الرجوع إليه لاحقاً والتنقل إليه من أي مكان أنت فيه — **مجاناً بالكامل**، بدون أي
اشتراك أو مفتاح API مدفوع.

## ✨ المزايا

- **حفظ الموقع الحالي بضغطة زر واحدة** (زر "حفظ موقعي الحالي" العائم بالأسفل).
- **تسمية كل موقع** باسم من اختيارك (البيت، العمل، بيت أبي...).
- **حفظ عدد غير محدود من الأماكن** في قاعدة بيانات محلية على جهازك.
- **مشاركة أي موقع** لأي تطبيق آخر مثبت (واتساب، تيليجرام، رسائل...) برابط خرائط قياسي.
- **التنقل/الإرشاد** إلى أي موقع محفوظ: يفتح تطبيق الخرائط المثبت لديك (خرائط جوجل أو غيره)
  ليرشدك إليه من موقعك الحالي.
- تصميم عصري بلغة Material 3، ويدعم الوضع الليلي تلقائياً.

## 🛠️ التقنيات المستخدمة (كلها مجانية)

| الغرض | التقنية |
|---|---|
| لغة البرمجة | **Kotlin** — اللغة الرسمية لأندرويد من جوجل، سريعة وآمنة وأقل عرضة للأخطاء |
| الواجهة | **Jetpack Compose** (Material 3) — أحدث وأسرع أداة لبناء واجهات أندرويد |
| تحديد الموقع | **Fused Location Provider** (خدمات جوجل بلاي) — مجاني تماماً، لا يحتاج مفتاح API |
| قاعدة البيانات | **Room** — تخزين محلي دائم على الجهاز، لا إنترنت مطلوب لحفظ الأماكن |
| الملاحة والمشاركة | **Android Intents** (geo: / google.navigation:) — تستخدم أي تطبيق خرائط مثبت لديك مجاناً، دون أي API خرائط مدفوع |

> لم يتم استخدام أي مفتاح Google Maps API مدفوع؛ التطبيق يعتمد على تطبيقات الخرائط
> المثبتة أصلاً على هاتفك (خرائط جوجل، Waze، أو غيرها) لعرض المسار والإرشاد.

## ▶️ طريقة التشغيل

1. ثبّت [Android Studio](https://developer.android.com/studio) (أحدث إصدار).
2. اختر **Open** وحدد مجلد `SavedPlaces` بعد فك ضغط الملف.
3. انتظر حتى تنتهي عملية **Gradle Sync** (قد تظهر رسالة لتحديث نسخة Gradle تلقائياً — وافق عليها).
4. وصّل جهاز أندرويد حقيقي (أو شغّل محاكي) واضغط **Run ▶**.
5. عند أول استخدام سيطلب التطبيق إذن الوصول للموقع — اسمح به حتى يعمل زر الحفظ.

> ملاحظة: لم يتم إرفاق ملفات `gradlew` / `gradle-wrapper.jar` الثنائية لأن بيئة الإنشاء هنا
> بدون إنترنت. عند فتح المشروع، ستقوم Android Studio تلقائياً بإعداد Gradle Wrapper، أو يمكنك
> تشغيل الأمر `gradle wrapper` إذا كان لديك Gradle مثبت على جهازك.

## ⬆️ رفع المشروع إلى GitHub

### الطريقة الأولى: عبر Git من سطر الأوامر
```bash
cd SavedPlaces
git init
git add .
git commit -m "أول إصدار من تطبيق أماكني"
```
ثم أنشئ مستودعاً جديداً وفارغاً من صفحة github.com (بدون إضافة README أو .gitignore عند
الإنشاء لتفادي التعارض)، وانسخ الأمرين اللذين يظهران تحت "…or push an existing repository
from the command line"، مثلاً:
```bash
git remote add origin https://github.com/USERNAME/SavedPlaces.git
git branch -M main
git push -u origin main
```

### الطريقة الثانية: بدون سطر أوامر (أسهل)
1. أنشئ مستودعاً جديداً في github.com.
2. من صفحة المستودع اضغط **Add file → Upload files**.
3. اسحب كل محتويات مجلد `SavedPlaces` (بعد فك الضغط) وأفلتها، ثم اضغط **Commit changes**.

### الطريقة الثالثة
استخدم برنامج **GitHub Desktop** (واجهة رسومية بسيطة) واختر "Add local repository" ثم انشر
المستودع من داخله دون كتابة أي أوامر.

## 🔐 الأذونات المطلوبة ولماذا

| الإذن | السبب |
|---|---|
| `ACCESS_FINE_LOCATION` / `ACCESS_COARSE_LOCATION` | لتحديد موقعك الحالي عند الضغط على زر الحفظ فقط |

لا يطلب التطبيق أي إذن إنترنت أو تخزين أو وصول لجهات الاتصال أو غيرها.

## 💡 أفكار لتطوير التطبيق مستقبلاً

- عرض خريطة مصغّرة (باستخدام OpenStreetMap المجاني) داخل التطبيق لمعاينة كل موقع.
- إمكانية تعديل اسم موقع محفوظ مسبقاً، أو تصنيف الأماكن ضمن مجلدات (منزل، عمل، مطاعم...).
- خاصية البحث والتصفية داخل قائمة الأماكن الطويلة.
- تصدير/استيراد الأماكن كملف واحد للنسخ الاحتياطي أو نقلها لجهاز آخر.
